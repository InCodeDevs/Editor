## Kurzfassung

Der Rufe Befehl kann dazu verwendet werden, eine Methode aufzurufen.

## Syntax

| Syntax                        | Bedeutung                                   |
| ----------------------------- | ------------------------------------------- |
| `Rufe die Methode <Name> auf` | `Ruft die Methode mit dem Namen <Name> auf` |
