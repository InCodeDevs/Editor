# Befehl: Erstelle

## Kurzfassung

Der Erstelle Befehl dient dazu neue Objekte, wie Knöpfe oder Überschriften, zu erstellen.
Außerdem kann der Erstelle Befehl dazu verwendet werden, eine neue Methode zu definieren.

## Syntax

| Syntax                        | Bedeutung                                      |
| ----------------------------- | ---------------------------------------------- |
| `Erstelle <Name>`             | Erstellt eine neue Variable ohne Typ und Wert. |
| `Erstelle <Name> als <Typ>`   | Erstellt eine neue Variable mit Typ.           |
| `Erstelle <Name> als Methode` | Erstellt eine neue Methode.                    |
