## Kurzfassung

Der Setze Befehl dient dazu eine Eigenschaft eines Objekts zu setzen.
Außerdem kann der Setze Befehl dazu verwendet werden, den Wert einer Variable zu verändern.

## Syntax

| Syntax                                                    | Bedeutung                                                             |
| --------------------------------------------------------- | --------------------------------------------------------------------- |
| `Setze [den/die/das] <Eigenschaft> von <Name> auf <Wert>` | `Setzt die <Eigenschaft> von der Variable <Name> auf den Wert <Wert>` |
