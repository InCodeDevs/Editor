## Kurzfassung

Der Wiederhole Befehl dient dazu einen oder mehrere Befehle beliebig mal zu wiederholen.
Außerdem kann er einen oder mehrere Befehle wiederholen, solange eine Bedingung erfüllt ist.

## Syntax

| Syntax                                                | Bedeutung                                                    |
| ----------------------------------------------------- | ------------------------------------------------------------ |
| `Wiederhole <Anzahl> mal`                             | `Wiederholt die Befehlsfolge <Anzahl> mal.`                  |
| `Wiederhole solange <Variable> <Operator> <Wert> ist` | `Wiederholt die Befehlsfolge solange Bedingung erfüllt ist.` |
