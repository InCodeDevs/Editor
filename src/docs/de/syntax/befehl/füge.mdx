## Kurzfassung

Der Füge Befehl dient dazu ein erstelltes Element zu einem bestehenden Element hinzuzufügen.
Dabei kann das Element auch zum Bildschirm hinzugefügt werden.

## Syntax

| Syntax                             | Bedeutung                                                                          |
| ---------------------------------- | ---------------------------------------------------------------------------------- |
| `Füge <Name> zu <Name2> hinzu`     | `Fügt das Element mit dem Namen <Name> zu dem Element mit dem Namen <Name2> hinzu` |
| `Füge <Name> zum Bildschirm hinzu` | `Fügt das Element mit dem Namen <Name> zum Bildschirm hinzu`                       |

## Beispiele

Erstellt das Element x und fügt es zum Bildschirm hinzu.
